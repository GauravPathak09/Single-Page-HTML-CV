# 🎨 Personal Portfolio (Styled with CSS)

A fully responsive personal portfolio website built to practice modern CSS layout techniques, typography, and color schemes. This project takes a previous HTML-only structure and brings it to life with styling.

**Project URL:** https://roadmap.sh/projects/portfolio-website

## 🎯 Project Features
* **Responsive Design:** Built using CSS Flexbox and the Box Model to ensure a structured layout.
* **Automatic Dark Mode:** Utilizes CSS Variables (`:root`) to automatically detect the user's system preferences and switch to a sleek dark theme.
* **Styled Contact Form:** A beautifully structured, centered contact card that seamlessly handles standard inputs and text areas.
* **Modern Typography:** Clean, readable text structure across all pages.

## 🛠️ Built With
* Semantic HTML5
* CSS3 (Flexbox, Variables, Media Queries)
