# ⏳ Changelog Component

A responsive, vertical timeline component built to display app updates, new features, and bug fixes. This project focuses on CSS positioning and layout techniques to create a clean, readable history log.

**Project URL:** https://roadmap.sh/projects/changelog-component

## 🎯 Features
* **Vertical Timeline:** Custom CSS borders and positioning to create a continuous timeline effect.
* **Semantic Structure:** Clean HTML grouping for dates, version numbers, and update notes.
* **Responsive Layout:** Adjusts spacing and alignment seamlessly for mobile screens.

## 🛠️ Built With
* HTML5
* CSS3 (Flexbox & Relative/Absolute Positioning)
